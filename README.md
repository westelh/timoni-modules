# timoni-modules

This repository contains a collection of Timoni modules for Kubernetes add-ons and integrations.

## Modules

- `acme-cluster-issuer`
- `acme-issuer`
- `ingress`
- `k8s-vault-addon`
- `letsencrypt-issuer`
- `oauth-vault`
- `oci-native-ingressclass`
- `oke-fss-storageclass`
- `selfsigned-issuer`
- `vault-connection`
- `vault-secret-sync`
- `vault-static-secret`

## Usage

Each module directory includes its own configuration and documentation. Refer to the module folder you want to deploy for details.
